# Mi Portafolio

Este es mi portafolio de servicios, una vitrina para mostrar mis proyectos más destacados y mis habilidades como desarrollador web.

## 🚀 Tecnologías y Arquitectura

El portafolio está construido con **React**, y ha sido migrado a **Next.js** para aprovechar el renderizado del lado del servidor (SSR) y las optimizaciones de rendimiento.

---
**NOTA:** El proyecto original fue creado con **Create React App (CRA)**. La migración a Next.js fue una decisión clave para mejorar el rendimiento, la escalabilidad y la SEO.
---

## ✨ Características Principales

-   **Rápido y optimizado:** Gracias a Next.js.
-   **Diseño Responsivo:** Se adapta a cualquier dispositivo.
-   **Proyectos dinámicos:** Muestra mis proyectos con detalles clave.

## 🛠️ Para ejecutar el proyecto localmente

Sigue estos pasos para ver el portafolio en tu máquina:

1.  Clona el repositorio:
    `git clone`
2.  Instala las dependencias:
    `cd [nombre-del-repositorio]`
    `npm install`
3.  Inicia el servidor de desarrollo:
    `npm run dev`
